三、完整章節複習內容

核心概念

  • Sequential representation：記憶體連續，random access O(1)，但中間 insert/delete 需要搬移 O(n)。
  • Linked representation：節點不必連續，靠 pointer 串接；若已知插入位置，insert/delete 可以 O(1)。
  • Singly list：每個 node 只有 link 指向下一個。
  • Doubly list：每個 node 有 left/right，可以雙向走訪。
  • Circular list：最後一個 node 指回 header 或 first，常用 header node 簡化空串列特例。

常考陷阱

  1. 刪除節點前，要先保存下一個指標。
  2. 插入節點時，順序通常是 newNode->link = current->link 再 current->link = newNode。
  3. Doubly delete 必改兩邊：x->left->right=x->right 與 x->right->left=x->left。

Heap

  • Heap = complete binary tree + heap order property。
  • Max heap：parent key ≥ child key，所以最大值在 root。
  • Min heap：parent key ≤ child key，所以最小值在 root。
  • Insert：新值放在最後一格，再往上比較交換。
  • Delete max：root 刪掉，最後一格補到 root，再往下交換。
  • Build heap：從最後一個 non-leaf node 開始 heapify。

DFS/BFS

  • DFS：往深處探索，可用遞迴或 stack。
  • BFS：由近到遠探索，用 queue，可解 unweighted shortest path。
  • Connected component：從未拜訪節點開始 DFS/BFS，每開始一次代表一個 component。

MST

  • Kruskal：把所有邊排序，由小到大選，不可形成 cycle；常搭配 disjoint set。
  • Prim：從任一點出發，每次選連到目前 tree 的最小邊。

Dijkstra

每次選尚未完成且距離最小的 vertex，更新它鄰居的距離。限制：edge weight 必須非負。

Bellman-Ford

可處理 negative edge，但可偵測 negative cycle。核心是重複 relax 所有 edges。

Floyd-Warshall

All-pairs shortest paths，三層迴圈，核心判斷：D[i][j] = min(D[i][j], D[i][k] + D[k][j])。

Topological Sort

只適用 DAG。常用 indegree = 0 queue 方法。

Critical Path

  1. 先做 topological order。
  2. 正向算 earliest start time。
  3. 反向算 latest start time。
  4. Slack = latest - earliest。
  5. Slack = 0 的活動在 critical path 上。

四、分級練習題

建議：第 1 週做 Level A，第 2 週做 Level B，第 3 週做 Level C。

Q1. 為什麼 linked list 在已知位置時 insert/delete 是 O(1)，但搜尋第 k 個元素是 O(n)？
因為 insert/delete 只改 pointer，不需要搬移元素；但 linked list 沒有 index，只能從 first 一個一個走到第 k 個。
Q2. BST 的 inorder traversal 會得到什麼結果？
會得到 key 的遞增排序，因為 BST 規則是 left < root < right。
Q3. BFS 使用哪一種資料結構？DFS 呢？
BFS 使用 queue；DFS 使用 recursion 或 stack。
Q4. 依序插入 20, 15, 30, 5, 40 到 Max Heap，畫出最後 heap。
最後 root 是 40。可能的 array 表示：[40,30,20,5,15]。插入 40 後一路 bubble up 到 root。
Q5. 對圖做 Topological Sort，若 indegree=0 有多個點，怎麼辦？
可以任選一個，因此 topological order 可能不唯一。考試通常接受符合所有先後限制的答案。
Q6. Hash table size=11，h(k)=k mod 11，插入 22, 33, 44，用 quadratic probing。
22→0；33 原本 0 衝突，0+1²=1；44 原本 0 衝突，試 1 也衝突，再試 0+2²=4，所以放 4。
Q7. 給一個 AOE network，要求 EST、LST、Critical Path。解題流程？
先 topological order；正向取 max 算 EST；反向取 min 算 LST；Slack = LST - EST；Slack=0 的邊組成 critical path。
Q8. Sorting 題問：「哪幾個 stable？哪幾個 in-place？」怎麼答？
Stable：Bubble、Insertion、Merge、Radix（若每一位使用 stable sort）。通常不 stable：Selection、Quick、Heap。In-place：Bubble、Insertion、Selection、Quick、Heap；Merge 通常不是。
Q9. Kruskal 和 Prim 最大差異？
Kruskal 從邊開始，排序所有 edges，避免 cycle；Prim 從一個 vertex 開始，逐步擴張 tree，每次選跨出去的最小 edge。

BFS

void BFS(int s){
    queue<int> q;
    visited[s] = true;
    q.push(s);
    while(!q.empty()){
        int u = q.front(); q.pop();
        cout << u << " ";
        for(int v : adj[u]){
            if(!visited[v]){
                visited[v] = true;
                q.push(v);
            }
        }
    }
}

Max Heap Push

void push(int x){
    heap.push_back(x);
    int i = heap.size() - 1;
    while(i > 1 && heap[i/2] < heap[i]){
        swap(heap[i/2], heap[i]);
        i /= 2;
    }
}

Dijkstra

vector<int> dijkstra(int s, int n){
    const int INF = 1e9;
    vector<int> dist(n, INF);
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> pq;
    dist[s] = 0;
    pq.push({0, s});
    while(!pq.empty()){
        auto [du, u] = pq.top(); pq.pop();
        if(du != dist[u]) continue;
        for(auto [v, w] : adj[u]){
            if(dist[u] + w < dist[v]){
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
    return dist;
}

Topological Sort

vector<int> topoSort(int n){
    queue<int> q;
    for(int i=0;i<n;i++) if(indeg[i]==0) q.push(i);
    vector<int> order;
    while(!q.empty()){
        int u=q.front(); q.pop();
        order.push_back(u);
        for(int v: adj[u]){
            if(--indeg[v]==0) q.push(v);
        }
    }
    return order;
}
