lec5_a Step-by-Step SVG C++ Generator

This version changes the original single-image / limited animation idea into topic-based step-by-step SVG frames.

How to run on Windows:
1. Open this folder.
2. Double-click ALL_IN_ONE_RUN_ALL_SVG.bat.
3. Open output folder. Each topic has its own folder, such as output/10_tree_traversals_steps/step_01.svg ...

No Graphviz / dot.exe is required. Every C++ program writes SVG directly.

Main change:
- Not fixed to only 6 frames.
- Each topic uses the number of steps required by that topic.
- Traversal, stack, queue, SAT, threaded tree and InsertRight contain more detailed frame-by-frame explanations.


[修正版說明]
本版強化 lec5a 每個下拉式主題的「動畫原理說明」，使 Tree Definition、Representation、Traversal、Expression Tree、Threaded Tree 等動畫說明更符合各自主題。其他 HTML 架構、CSS、JavaScript、SVG、程式碼、中文註解、輸出結果、函式與變數說明皆保留。
