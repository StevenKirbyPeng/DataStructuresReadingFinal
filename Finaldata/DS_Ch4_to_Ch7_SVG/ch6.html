多步驟 SVG 分解講解

Step 1：AOV network 是 DAG
正確：topological sort 不是數字排序。

正確：只有 DAG 可完成。
Step 3：輸出並刪除 outgoing edges
正確：indegree=0 表示無前置工作。

詳細中文解說

Topological sort 用於 DAG。AOV network 中 vertex 表 activity，edge 表先後關係；若有 cycle 則無法做拓樸排序。常見做法是計算 indegree，每次取 indegree=0 的點輸出，再把它的 outgoing edges 移除。Critical path 題目會再加上 duration，計算 earliest start time 與 latest start time，slack=0 的 activity 形成 critical path。

Chapter 6 GraphsBellman-Ford / Floyd / Transitive Closure

多步驟 SVG 分解講解

Step 1：Bellman-Ford 初始化距離
正確：Bellman-Ford 可處理負權但不可有負環。
Step 2：所有邊重複 relax |V|-1 次
正確：|V|-1 次足以涵蓋簡單路徑。
Step 3：再檢查負環
正確：仍可改善代表負環。
Step 4：Floyd 用中繼點 k 更新 all-pairs
正確：Floyd 是 all-pairs shortest paths。
Step 5：Transitive closure 判斷可達性
正確：transitive closure 是 reachability。

詳細中文解說

Bellman-Ford 可處理負權重邊，做 |V|-1 輪 relaxation，並可偵測 negative cycle。Floyd-Warshall 解 all-pairs shortest paths，核心是逐步允許中繼點 k，更新 d[i][j]。Transitive closure 關心可不可達，不一定關心距離，常用 Warshall 演算法把 reachability matrix 更新。這些主題與 Dijkstra 的差別是：Dijkstra 快但要求非負權重；Floyd 較適合 all pairs；transitive closure 是布林可達性。
