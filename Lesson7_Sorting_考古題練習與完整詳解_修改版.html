題型 1：Bubble Sort 追蹤題

英文題目

Given 5, 4, 1, 2, 8, show the array after each pass of Bubble Sort.

演算法原理講解

Bubble Sort 每一輪從左到右比較相鄰兩個元素，若順序錯誤就交換。第 1 輪會把最大值推到最右邊，第 2 輪把次大值推到倒數第二格。

題型 2：Selection Sort 追蹤與穩定性

英文題目

Sort 8, 5, 2, 6, 9, 3, 1 by Selection Sort. Show each pass and explain whether it is stable.

演算法原理講解

Selection Sort 每輪在未排序區間找最小值，和未排序區間第一格交換。它交換次數少，但是通常不是 stable。

題型 3：Insertion Sort 最佳/最壞情況

英文題目

Apply Insertion Sort to 5, 4, 3, 2, 1. Show the result after each insertion.

演算法原理講解

把第 j 個元素插入前面已排序的子陣列。反向排序是最壞情況，每次都要搬移很多元素；已排序是最佳情況，只比較一次就停止。

j=3: 2 3 4 5 1
j=4: 1 2 3 4 5
Best case: O(n), Worst case: O(n^2), Stable: Yes, In-place: Yes

key目前要插入的值。

i從已排序區間尾端往左掃描。
insertionSort逐步擴大已排序區間。
題型 4：Quick Sort Pivot 分割

英文題目

Use Quick Sort with the leftmost element as pivot to sort 26, 5, 37, 1, 61, 11, 59, 15, 48, 19. Show the partition process.

演算法原理講解

Quick Sort 採 divide-and-conquer。先選 pivot，把小於等於 pivot 的放左邊，大於等於 pivot 的放右邊，再遞迴排序左右子陣列。

題型 7：Decision Tree 證明題

英文題目

Prove that every comparison-based sorting algorithm has worst-case lower bound Ω(n log n).

演算法原理講解

比較排序只能透過兩兩比較得到 True/False，因此整個排序過程可以用二元 decision tree 表示。每個葉節點代表一種排序結果。

題型 8：Heap Sort 改成遞減排序

英文題目

Modify HeapSort so that the result is in nonincreasing order.

演算法原理講解

原本 max heap 每次把最大值放到最後，因此結果是遞增。若要遞減，就改建 min heap，每次把最小值放到最後。
