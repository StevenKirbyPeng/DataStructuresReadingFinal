中文解析

  1. 陣列中間插入：必須先把 HAT 往後搬，空出 index 2。

Question

Write a C++ function to reverse a singly linked list in-place. Then trace the list 10 → 20 → 30 → 40 after the function finishes.

中文翻譯

請寫出 C++ 函式，將單向鏈結串列原地反轉，並追蹤 10 → 20 → 30 → 40 反轉後的結果。

反轉動畫

完整 C++ 程式碼

#include <iostream>
using namespace std;

struct Node {
    int data;       // 節點資料
    Node* link;     // 指向下一個節點
    Node(int d, Node* next = nullptr) : data(d), link(next) {}
};

// 函式功能：將 first 所指向的單向鏈結串列原地反轉
// 參數：first 是串列第一個節點的指標，使用參考傳入，才能更新真正的表頭
// 時間複雜度：O(n)，每個節點處理一次
// 空間複雜度：O(1)，只使用 current、previous、r 三個指標
void Reverse(Node*& first) {
    Node* current = first;      // current 指向目前正在處理的節點
    Node* previous = nullptr;   // previous 指向已反轉部分的第一個節點

    while (current != nullptr) {
        Node* r = previous;           // r 暫存 previous
        previous = current;           // previous 前進到 current
        current = current->link;      // current 先移到下一個節點
        previous->link = r;           // 將目前節點 link 反轉回去
    }

    first = previous; // previous 最後會停在原本最後一個節點，成為新的 first
}

void Print(Node* p) {
    while (p) {
        cout << p->data;
        if (p->link) cout << " -> ";
        p = p->link;
    }
    cout << endl;
}

int main() {
    Node* first = new Node(10, new Node(20, new Node(30, new Node(40))));
    Reverse(first);
    Print(first); // 40 -> 30 -> 20 -> 10
    return 0;
}

函式說明

Reverse(Node*& first)
原地反轉整條 singly linked list，最後更新 first。
current
目前正在處理的節點。
previous
已經反轉好的前半段開頭。
r
暫存 previous，避免 link 改掉後遺失。
Answer：反轉後為 40 → 30 → 20 → 10。

Question

Given this binary tree, write the output of inorder, preorder, postorder, and level-order traversal.
        A
       / \
      B   C
     / \ / \
    D  I F  G

中文翻譯

請寫出這棵二元樹的 inorder、preorder、postorder 與 level-order 結果，並提供 C++ 參考程式。

走訪動畫

完整 C++ 程式碼

#include <iostream>
#include <queue>
using namespace std;

struct Node {
    char data;       // 節點資料
    Node* left;      // 左子樹
    Node* right;     // 右子樹
    Node(char d) : data(d), left(nullptr), right(nullptr) {}
};

void inorder(Node* root) {       // LVR：左、根、右
    if (!root) return;
    inorder(root->left);
    cout << root->data << ' ';
    inorder(root->right);
}

void preorder(Node* root) {      // VLR：根、左、右
    if (!root) return;
    cout << root->data << ' ';
    preorder(root->left);
    preorder(root->right);
}

void postorder(Node* root) {     // LRV：左、右、根
    if (!root) return;
    postorder(root->left);
    postorder(root->right);
    cout << root->data << ' ';
}

void levelorder(Node* root) {    // 使用 queue 做 level-order
    if (!root) return;
    queue<Node*> q;
    q.push(root);
    while (!q.empty()) {
        Node* cur = q.front(); q.pop();
        cout << cur->data << ' ';
        if (cur->left) q.push(cur->left);
        if (cur->right) q.push(cur->right);
    }
}

函式說明

inorder
左子樹 → 根 → 右子樹。
preorder
根 → 左子樹 → 右子樹。
postorder
左子樹 → 右子樹 → 根。
levelorder
使用 queue 逐層拜訪。
Answer：Inorder: D B I A F C G；Preorder: A B D I C F G；Postorder: D I B F G C A；Level-order: A B C D I F G.

Question

Use the given Adjust and HeapSort idea to sort [4,1,3,2,16,9,10,14,8,7] into nondecreasing order. Provide commented C++ code and the final output.

中文翻譯

使用 Adjust 與 HeapSort 的概念，把陣列 [4,1,3,2,16,9,10,14,8,7] 排成遞增順序，提供有中文註解的 C++ 程式與最後輸出。

Heap Sort 動畫

完整 C++ 程式碼

#include <iostream>
#include <algorithm>
using namespace std;

// 對 root 位置做 max heapify，heap 範圍是 a[1] 到 a[n]
void Adjust(int a[], int root, int n) {
    int e = a[root];             // 暫存 root 的值
    int j = 2 * root;            // j 指向左孩子

    for (; j <= n; j *= 2) {
        // 如果右孩子存在且比左孩子大，改選右孩子
        if (j < n && a[j] < a[j + 1]) j++;

        // 如果 e 已經比最大孩子大，可以放在目前位置
        if (e >= a[j]) break;

        // 否則把較大的 child 往上提
        a[j / 2] = a[j];
    }

    // 把 e 放到最後找到的位置
    a[j / 2] = e;
}

void HeapSort(int a[], int n) {
    // 1. bottom-up 建立 max heap
    for (int i = n / 2; i >= 1; --i) {
        Adjust(a, i, n);
    }

    // 2. 每次把最大值 a[1] 換到最後，再縮小 heap
    for (int i = n - 1; i >= 1; --i) {
        swap(a[1], a[i + 1]);
        Adjust(a, 1, i);
    }
}

int main() {
    int a[11] = {0, 4, 1, 3, 2, 16, 9, 10, 14, 8, 7}; // a[0] 不用
    HeapSort(a, 10);
    for (int i = 1; i <= 10; ++i) cout << a[i] << ' ';
    return 0;
}

函式說明

Adjust
維護以 root 為根的 max heap 性質。
HeapSort
先建 max heap，再反覆把最大值交換到右側排序區。
為什麼 a[0] 不用
投影片採用 heap[1..n]，parent/child index 公式較簡單。
Answer：Output: 1 2 3 4 7 8 9 10 14 16. Time complexity: O(n log n).

Question

Given BST keys: 6, 3, 15, 12, 10, 7.
a. Delete key 6 using its inorder successor.
b. If each node stores leftSize = 1 + number of nodes in its left subtree, explain how RankGet(4) works.

中文翻譯

給定 BST key：6,3,15,12,10,7。a. 使用 inorder successor 刪除 6。b. 如果每個節點存 leftSize，說明 RankGet(4) 如何找到第 4 小。

圖解

參考 C++ 程式碼：RankGet

// 搜尋第 r 小的元素，假設每個節點都有 leftSize 欄位
// leftSize = 1 + 左子樹節點數，也就是目前 root 在自己子樹中的排名
template <class K, class E>
pair<K, E>* BST<K, E>::RankGet(int r) {
    TreeNode<pair<K, E>>* p = root; // 從 root 開始

    while (p) {
        if (r < p->leftSize) {
            // 第 r 小在左子樹
            p = p->leftChild;
        } else if (r > p->leftSize) {
            // 第 r 小在右子樹，扣掉左子樹與目前節點
            r -= p->leftSize;
            p = p->rightChild;
        } else {
            // r 剛好等於 leftSize，目前節點就是第 r 小
            return &p->data;
        }
    }
    return nullptr; // 找不到
}

中文解析

  1. 刪除有兩個 children 的節點時，可用 successor，也就是右子樹最小值。
  2. 6 的右子樹中最小值是 7，所以用 7 取代 6。
  3. RankGet 每次利用 leftSize 判斷第 r 小在左邊、目前節點或右邊。
Answer：Delete 6 by successor: successor is 7, so 7 replaces 6; original 7 node is removed from the left side of 10. Sorted order is 3,6,7,10,12,15 before deletion, so the 4th smallest before deletion is 10.

中文翻譯

DFS/BFS 動畫

完整 C++ 程式碼

#include <iostream>
#include <vector>
#include <queue>
using namespace std;

void DFS(int v, const vector<vector<int>>& adj, vector<bool>& visited) {
    visited[v] = true;          // 標記目前節點已拜訪
    cout << v << ' ';          // 輸出 DFS 拜訪順序
    for (int w : adj[v]) {      // 依照 adjacency list 的順序拜訪鄰居
        if (!visited[w]) DFS(w, adj, visited);
    }
}

void BFS(int start, const vector<vector<int>>& adj) {
    vector<bool> visited(adj.size(), false);
    queue<int> q;
    visited[start] = true;
    q.push(start);

    while (!q.empty()) {
        int v = q.front(); q.pop();
        cout << v << ' ';
        for (int w : adj[v]) {
            if (!visited[w]) {
                visited[w] = true; // 入隊前就標記，避免重複入隊
                q.push(w);
            }
        }
    }
}

int main() {
    vector<vector<int>> adj = {
        {1,2}, {0,3,4}, {0,5,6}, {1,7},
        {1,7}, {2,7}, {2,7}, {3,4,5,6}
    };
    vector<bool> visited(8, false);
    cout << "DFS: "; DFS(0, adj, visited); cout << endl;
    cout << "BFS: "; BFS(0, adj); cout << endl;
}

中文解析

DFS 像 preorder，一路走到底；BFS 像 level-order，使用 queue 逐層擴散。

Answer：DFS: 0, 1, 3, 7, 4, 5, 2, 6. BFS: 0, 1, 2, 3, 4, 5, 6, 7.

Question

Given weighted edges:
(0,5,10), (2,3,12), (1,6,14), (1,2,16), (3,6,18), (3,4,22), (4,6,24), (4,5,25), (0,1,28).
a. Use Kruskal's algorithm to find the MST.
b. Explain one difference between Kruskal and Prim.

中文翻譯

給定 weighted edges，使用 Kruskal 找 MST，並說明 Kruskal 與 Prim 的差異。

MST 動畫

中文解析

  1. Kruskal：先把所有邊依 weight 排序。
  2. 由小到大選邊，但不能形成 cycle。
  3. 使用 Union-Find 判斷兩端點是否已在同一集合。
  4. n=7 個 vertices，所以 MST 需要 6 條邊。
Answer：MST edges: (0,5,10), (2,3,12), (1,6,14), (1,2,16), (3,4,22), (4,5,25). Total cost = 99. Kruskal grows a forest by adding globally smallest safe edges; Prim grows one tree from a starting vertex by adding the cheapest edge leaving the current tree.
