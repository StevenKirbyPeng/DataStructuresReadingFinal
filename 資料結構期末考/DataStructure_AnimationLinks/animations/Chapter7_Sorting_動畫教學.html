01｜Sorting 基本觀念：In-place、Stable、Internal / External

核心說明

Sorting 的輸入是一串 records，每個 record 有 key；輸出是 key 依非遞減順序排列的 permutation。若相同 key 在輸出中仍保持原本先後順序，稱為 stable sorting。若除了工作陣列外只使用常數個額外變數，稱為 in-place sorting。

考試重點：題目常問某排序是否 stable、是否 in-place、best / worst / average time complexity。
常見錯誤：stable 不是「排序結果唯一」，而是「相同 key 的原始相對順序不變」。

02｜Bubble Sort：相鄰比較與 bubble up

原理

Bubble Sort 每次比較相鄰兩個元素，若順序錯就交換。第一輪結束後，最大值會被推到最右邊。

時間複雜度：一般版本 worst / average 為 O(n²)。可用 swapped 旗標讓 best case 變 O(n)。
易錯：每一 pass 的比較範圍會縮小，因為右邊已排序。
準備中
Bubble Sort C++
template <class T>
void BubbleSort(T* a, int n) {
    for (int j = n; j > 1; --j) {
        for (int k = 1; k < j; ++k) {
            if (a[k+1] < a[k]) swap(a[k+1], a[k]);
        }
    }
}
03｜Selection Sort：找最小值放到未排序區第一格

原理

Selection Sort 每一輪從未排序區找出最小值，和未排序區第一個位置交換。左側逐漸形成 sorted prefix。

特性：比較次數固定約 n(n-1)/2，交換次數少，通常不是 stable。
準備中
04｜Insertion Sort：把第 j 個元素插入左側已排序串列

原理

Insertion Sort 將 a[j] 暫存於 a[0]，從 j-1 往左比較，只要比暫存值大就右移，最後把暫存值放到空位。

特性：stable、in-place；best case 已排序時 O(n)，worst case 反序時 O(n²)。
投影片重點：a[0] 是 temporary sentinel，用來保存 a[j]。
準備中
05｜Quick Sort Partition：pivot、i、j 指標交換

原理

Quick Sort 選 pivot，讓左邊元素 ≤ pivot、右邊元素 ≥ pivot。投影片程式以 leftmost value 作 pivot，用 i 從左找大、j 從右找小，交換直到 i ≥ j，再把 pivot 與 a[j] 交換。

易錯：Quick Sort 的 combine 不需要做事；主要工作在 partition。
準備中
06｜Quick Sort 遞迴分割：left / right sublists

原理

Partition 後 pivot 固定在正確位置，左右子串列再分別遞迴排序，左、右子串列之間不再比較。

複雜度：best / average O(n log n)，worst O(n²)。若每次 pivot 都是 median，最接近平衡。
準備中
07｜Decision Tree：比較排序法 Ω(n log n) 下界

原理

只靠 comparison 的排序法，每一個比較只有 Yes / No 兩種結果，所以可用 binary decision tree 表示。n 個不同元素有 n! 種排列，因此 decision tree 至少要有 n! 個 leaves，高度至少 log₂(n!) + 1。

結論：任何 comparison sort 的 worst-case time complexity 是 Ω(n log n)。
準備中
08｜Iterative Merge Sort：長度 1、2、4... 成對合併

原理

把 n 個長度 1 的 sorted lists 成對合併成長度 2，再合併成長度 4，直到只剩一個 sorted list。

特性：stable，時間 O(n log n)，需要額外陣列，通常不是 in-place。
準備中
09｜Merging：兩個已排序子串列合併

原理

Merge 的兩個輸入串列都已排序。每次比較 i1 與 i2 所指元素，較小者放到 mergedList 的 iResult，直到其中一邊用完，再把剩下元素複製過去。

穩定性：若相等時選左邊 initList[i1]，就能保持 stable。
準備中
10｜Recursive Merge Sort：Divide → Conquer → Combine

原理

Recursive Merge Sort 先把陣列分成左右兩半，遞迴排序兩邊，再 merge 回一個排序好的串列。

遞迴式：T(n)=2T(n/2)+O(n)，所以 T(n)=O(n log n)。

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void print(const vector<int>& a){ for(int x:a) cout << x << ' '; cout << '\n'; }

void bubbleSort(vector<int> a){
    cout << "Bubble Sort:\n";
    for(int j=a.size()-1;j>0;--j){
        for(int k=0;k<j;++k) if(a[k+1]<a[k]) swap(a[k+1],a[k]);
        print(a);
    }
}

void selectionSort(vector<int> a){
    cout << "Selection Sort:\n";
    for(int i=0;i+1<(int)a.size();++i){
        int mn=i;
        for(int j=i+1;j<(int)a.size();++j) if(a[j]<a[mn]) mn=j;
        swap(a[i],a[mn]);
        print(a);
    }
}

void insertionSort(vector<int> a){
    cout << "Insertion Sort:\n";
    for(int j=1;j<(int)a.size();++j){
        int temp=a[j], i=j-1;
        while(i>=0 && temp<a[i]){ a[i+1]=a[i]; --i; }
        a[i+1]=temp;
        print(a);
    }
}

int partitionQS(vector<int>& a,int left,int right){
    int pivot=a[left], i=left+1, j=right;
    while(true){
        while(i<=right && a[i]<pivot) ++i;
        while(j>=left+1 && a[j]>pivot) --j;
        if(i>=j) break;
        swap(a[i],a[j]); ++i; --j;
    }
    swap(a[left],a[j]);
    return j;
}
void quickSort(vector<int>& a,int l,int r){
    if(l<r){ int p=partitionQS(a,l,r); quickSort(a,l,p-1); quickSort(a,p+1,r); }
}

void mergeSort(vector<int>& a,int l,int r){
    if(l>=r) return;
    int m=(l+r)/2;
    mergeSort(a,l,m); mergeSort(a,m+1,r);
    vector<int> tmp;
    int i=l,j=m+1;
    while(i<=m && j<=r){
        if(a[i]<=a[j]) tmp.push_back(a[i++]); // <= keeps stability
        else tmp.push_back(a[j++]);
    }
    while(i<=m) tmp.push_back(a[i++]);
    while(j<=r) tmp.push_back(a[j++]);
    for(int k=0;k<(int)tmp.size();++k) a[l+k]=tmp[k];
}

int main(){
    vector<int> a={5,4,1,2,8};
    bubbleSort(a); selectionSort(a); insertionSort(a);
    vector<int> q={26,5,37,1,61,11,59,15,48,19};
    quickSort(q,0,q.size()-1); cout << "Quick Sort result: "; print(q);
    vector<int> m={38,27,43,3,9,82,10};
    mergeSort(m,0,m.size()-1); cout << "Merge Sort result: "; print(m);
    return 0;
}
