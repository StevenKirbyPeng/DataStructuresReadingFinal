題目原文

Define a tree recursively. Why are recursive algorithms natural for trees?

中文翻譯

請用遞迴方式定義 tree，並說明為什麼 tree 很適合用 recursive algorithms。

核心觀念

treerootsubtreerecursive definition

動態圖解

解題步驟

  1. Tree 是階層式資料結構。
  2. 一棵 tree 有一個 root。
  3. Root 以外的節點可分成互不相交的 subtree。
  4. 因為 subtree 本身也是 tree，所以處理 tree 很自然會使用 recursion。
完整答案：A tree is a finite set of one or more nodes. There is one distinguished node called the root. The remaining nodes are partitioned into zero or more disjoint sets, and each set is itself a tree, called a subtree of the root. Since each subtree has the same structure as the original tree, recursive algorithms naturally match the recursive definition of trees.
考試容易錯：一般 tree 至少有一個節點；但 binary tree 可以是 empty，這點後面常考。

題目原文

Given the following tree, answer the questions.

        A
      / | \
     B  C  D
    / \ | /|\
   E  F G H I J
  / \     |
 K   L    M

a. What is the root?
b. What are the leaves?
c. What is degree(A), degree(C), and degree(F)?
d. What are the children of D?
e. What are the ancestors of M?
f. What is the height of the tree if the root is at level 1?

中文翻譯

根據圖中的一般樹，回答 root、leaf、degree、children、ancestors 與 height。

核心觀念

rootdegreeleafchildrenancestorsheight

動態圖解

解題步驟

  1. Root 是最上方節點 A。
  2. Leaf 是 degree 0 的節點：K、L、F、G、M、I、J。
  3. A 有 B、C、D 三個 children；C 有 G 一個 child；F 沒有 child。
  4. D 的 children 是 H、I、J。
  5. M 的 ancestors 是 A、D、H。
  6. Root 在 level 1，最深路徑 A→D→H→M 為 level 4，所以 height=4。
完整答案：a. A。b. K, L, F, G, M, I, J。c. degree(A)=3, degree(C)=1, degree(F)=0。d. H, I, J。e. A, D, H。f. 4。
考試容易錯：leaf 不是「畫在最下面」才算；F、G 雖然比較上層，但沒有小孩，所以也是 leaf。

parenthesizationroot firstsubtree list

題目原文

For a k-ary tree with n nodes, how many child fields are NULL? Prove your answer. Then compute the number of NULL child fields when k = 4 and n = 10.

中文翻譯

一棵有 n 個節點的 k 元樹，每個節點有 k 個 child fields，請問有幾個 child fields 是 NULL？請證明，並計算 k=4、n=10 時的答案。

核心觀念

k-ary treechild fieldsproofwasted memory

圖解說明

解題步驟

  1. 每個 node 有 k 個 child fields，所以總 child fields = nk。
  2. 除了 root 以外，每個節點都會被某一個 child field 指到。
  3. 因此有使用的 child fields = n−1。
  4. NULL child fields = nk − (n−1) = n(k−1)+1。
  5. k=4, n=10：NULL fields = 10×3+1 = 31。
完整答案：There are n(k−1)+1 NULL child fields. When k=4 and n=10, the number is 31.
考試容易錯：不能只算 leaf 的 child fields，因為 degree 小於 k 的 nonleaf nodes 也有空的 child fields。

題目原文

Convert the following general tree into a left-child right-sibling representation, and then draw the corresponding degree-two tree.

        A
      / | \
     B  C  D
    / \   / \
   E   F H   I

中文翻譯

請把一般樹轉成 left-child right-sibling 表示法，再畫成 degree-two tree。

核心觀念

left childright siblingdegree-two treebinary representation

動態圖解

解題步驟

  1. 每個節點的 left child 指向它最左邊的 child。
  2. 同一層兄弟節點用 right sibling 串起來。
  3. A.left=B；B.right=C；C.right=D。
  4. B.left=E；E.right=F。
  5. D.left=H；H.right=I。
  6. Degree-two tree 可視為把 right sibling 線順時針旋轉 45 度成 right child。
完整答案：A 的 left child 為 B；B 的 right sibling 為 C；C 的 right sibling 為 D；B 的 left child 為 E；E 的 right sibling 為 F；D 的 left child 為 H；H 的 right sibling 為 I。轉為 degree-two tree 後，left child 保留為左子，right sibling 變為右子。
考試容易錯：Root 的 right child 在轉換後永遠是空，因為 root 沒有 right sibling。

題目原文

Answer the following questions.
a. Can a binary tree be empty?
b. Is a binary tree a special case of a general tree? Explain.
c. Classify the three trees shown in the animation as skewed, complete, or full.

中文翻譯

請回答二元樹是否可以為空、二元樹是否為一般樹的特例，並判斷 skewed、complete、full。

核心觀念

binary treeempty treeleft/right distinctionskewedcompletefull

題目原文

For a nonempty binary tree, prove that if n0 is the number of leaf nodes and n2 is the number of nodes of degree 2, then n0 = n2 + 1. Also answer: what is the maximum number of nodes in a binary tree of depth k?

中文翻譯

請證明非空二元樹中 leaf nodes 數量 n0 與 degree-2 nodes 數量 n2 滿足 n0=n2+1。並回答深度 k 的二元樹最多有幾個節點。

核心觀念

n0 = n2 + 1branchesmaximum nodesproof

公式圖解

解題步驟

  1. 令 n1 為 degree 1 的節點數。
  2. 總節點數 n = n0 + n1 + n2。
  3. 除了 root 外，每個節點都有一條 branch 連入，所以 B = n−1。
  4. 由節點往外長出的 branch 數量為 n1 + 2n2。
  5. 所以 n0+n1+n2−1 = n1+2n2，化簡得 n0=n2+1。
  6. 深度 k 的二元樹最多節點數是 1+2+...+2^(k−1)=2^k−1。
完整答案：n0 = n2 + 1。Maximum number of nodes at depth k is 2k − 1。
考試容易錯：Branch 數不是 n，而是 n−1，因為 root 沒有 incoming branch。

題目原文

A complete binary tree is stored in an array from index 1 to n. For node i, give the index of its parent, left child, and right child. If n=10 and i=5, what are the parent, left child, and right child?

中文翻譯

完全二元樹用陣列 index 1 到 n 儲存。請寫出 i 的 parent、left child、right child 位置。若 n=10 且 i=5，答案是什麼？

核心觀念

complete binary treearray indexparentchildren

圖解說明

解題步驟

  1. Index 從 1 開始。
  2. parent(i)=⌊i/2⌋，但 i=1 是 root，沒有 parent。
  3. leftChild(i)=2i，若 2i>n 則不存在。
  4. rightChild(i)=2i+1，若 2i+1>n 則不存在。
  5. n=10, i=5：parent=2，leftChild=10，rightChild=11>10，所以沒有右小孩。
完整答案：parent(5)=2；leftChild(5)=10；rightChild(5) 不存在，因為 11>n。
考試容易錯：不要只算 2i+1=11 就寫 right child 是 11，還要檢查是否超過 n。

題目原文

Explain the iterative inorder traversal algorithm using a stack. Give the time and space complexity, and provide a C++ implementation with comments.

中文翻譯

請解釋用 stack 做非遞迴 inorder traversal 的演算法，給出時間與空間複雜度，並提供有註解的 C++ 程式。

核心觀念

iterative inorderstackO(n)space O(h)

演算法動畫

完整 C++ 程式碼

#include <iostream>
#include <stack>
using namespace std;

// 二元樹節點結構
struct TreeNode {
    char data;              // 節點資料
    TreeNode* leftChild;    // 左子樹指標
    TreeNode* rightChild;   // 右子樹指標

    TreeNode(char x) : data(x), leftChild(nullptr), rightChild(nullptr) {}
};

// 非遞迴中序走訪：Left → Visit → Right
void NonrecInorder(TreeNode* root) {
    stack<TreeNode*> s;       // 用 stack 暫存尚未拜訪的祖先節點
    TreeNode* current = root; // 從 root 開始

    while (current != nullptr || !s.empty()) {
        // 1. 一路往左走，沿途把節點放進 stack
        while (current != nullptr) {
            s.push(current);
            current = current->leftChild;
        }

        // 2. 左邊走到底後，取出 stack top 並拜訪
        current = s.top();
        s.pop();
        cout << current->data << ' ';

        // 3. 改往右子樹走，下一輪又會先走右子樹的最左邊
        current = current->rightChild;
    }
}

int main() {
    TreeNode* root = new TreeNode('A');
    root->leftChild = new TreeNode('B');
    root->rightChild = new TreeNode('C');
    root->leftChild->leftChild = new TreeNode('D');
    root->leftChild->leftChild->rightChild = new TreeNode('I');
    root->rightChild->leftChild = new TreeNode('F');
    root->rightChild->rightChild = new TreeNode('G');

    cout << "Inorder: ";
    NonrecInorder(root); // output: D I B A F C G
    return 0;
}

函式與語法說明

TreeNode
二元樹節點，包含 data、leftChild、rightChild。
stack<TreeNode*>
儲存還沒 visit 的祖先節點。
current
目前走訪到的節點。
內層 while
一路往左走到底。
s.top() / s.pop()
取出下一個應該 visit 的節點。

解題步驟

  1. 從 root 開始，一路走 leftChild，沿途 push 進 stack。
  2. 走到 null 後，pop 一個節點並 visit。
  3. 接著改走該節點的 rightChild。
  4. 重複直到 current 為 null 且 stack 為空。
完整答案：Time complexity = O(n)，每個節點 push/pop 各一次。Space complexity = O(h)，h 是樹高，最壞可為 O(n)。
考試容易錯：空間複雜度精確說是 O(h)，不是永遠 O(n)；只有 skewed tree 最壞情況才是 O(n)。

題目原文

Why can postorder traversal be used to evaluate an expression tree? Given the expression tree for (x1 AND NOT x2) OR x3, evaluate it when x1=true, x2=false, and x3=false.

中文翻譯

為什麼 expression tree 可用 postorder traversal 評估？給定運算式 (x1 AND NOT x2) OR x3，當 x1=true、x2=false、x3=false 時，請求值。

核心觀念

SATexpression treepostorderoperatoroperand

求值動畫

解題步驟

  1. Expression tree 中，leaf 是 operand，nonleaf 是 operator。
  2. Operator 必須等左右子運算式都有值後才能計算。
  3. Postorder 是 Left → Right → Root，剛好先算子樹，再算 operator。
  4. x1=true，x2=false，所以 NOT x2 = true。
  5. x1 AND NOT x2 = true AND true = true。
  6. true OR x3(false) = true。
完整答案：Postorder traversal is used because it evaluates the operands or subexpressions before the operator. The value of (x1 AND NOT x2) OR x3 is true.
考試容易錯：NOT 通常只有右子樹或單一 operand；不要把 NOT 當成二元 operator。

中文翻譯

根據二元樹與 inorder sequence，回答 threaded binary tree 的空鏈結數量、inorder successor、Next() 的找法，以及 InsertRight(s,r) 的意義。

核心觀念

threaded binary treeNULL linksinorder successorrightThreadInsertRight

Thread 動態圖解

完整 C++ 程式碼

// Threaded binary tree 的節點結構
// leftThread / rightThread 用來判斷 leftChild / rightChild 是真正 child 還是 thread
template <class T>
struct ThreadedNode {
    bool leftThread;              // true 表示 leftChild 是 inorder predecessor thread
    ThreadedNode<T>* leftChild;
    T data;
    ThreadedNode<T>* rightChild;
    bool rightThread;             // true 表示 rightChild 是 inorder successor thread
};

// 找 currentNode 的 inorder successor
template <class T>
T* ThreadInorderIterator<T>::Next() {
    ThreadedNode<T>* temp = currentNode->rightChild;

    // 如果 rightThread == false，代表右邊是真正的右子樹
    // inorder successor 是右子樹中最左邊的節點
    if (!currentNode->rightThread) {
        while (!temp->leftThread) {
            temp = temp->leftChild;
        }
    }

    currentNode = temp;

    // 若回到 header/root sentinel，代表 traversal 結束
    if (currentNode == root) return nullptr;
    return &currentNode->data;
}

// 將 r 插入成為 s 的右兒子節點
template <class T>
void ThreadTree<T>::InsertRight(ThreadedNode<T>* s, ThreadedNode<T>* r) {
    // r 繼承 s 原本的右指標與 rightThread 狀態
    r->rightChild = s->rightChild;
    r->rightThread = s->rightThread;

    // r 的 inorder predecessor 是 s
    r->leftChild = s;
    r->leftThread = true;

    // s 的右兒子改成 r，且不再是 thread
    s->rightChild = r;
    s->rightThread = false;

    // 如果 r 右邊接的是原本 s 的右子樹，需修正該子樹最左節點的 predecessor thread
    if (!r->rightThread) {
        ThreadedNode<T>* temp = InorderSucc(r);
        temp->leftChild = r;
    }
}

函式與語法說明

leftThread
true 表示 leftChild 不是左子，而是 inorder predecessor thread。
rightThread
true 表示 rightChild 不是右子，而是 inorder successor thread。
ThreadInorderIterator::Next()
不用 stack，以 thread 或右子樹最左節點找下一個 inorder 節點。
InsertRight(s,r)
把 r 插入成 s 的右 child，並修正 successor / predecessor thread。

解題步驟

  1. Binary tree 有 n 個節點，共有 2n 個 child links。
  2. 真正使用的 links 是 n−1，所以 NULL links = 2n−(n−1)=n+1。
  3. Inorder sequence 是 D I B A F C G，所以 successor(B)=A，successor(G)=header/end。
  4. 若 p 有 right thread，thread 直接指向 successor。
  5. 若 p 沒有 right thread，successor 是 p 的右子樹中最左邊的節點。
  6. InsertRight(s,r) 是把 r 加成 s 的右兒子，並保持 threaded binary tree 的 thread 正確。
完整答案：a. n+1。b. successor(B)=A，successor(G)=header 或 traversal end。c. 若 rightThread=true，直接沿 rightChild；若 rightThread=false，走到右子樹最左節點。d. InsertRight(s,r) 將 r 插入為 s 的右 child，並調整 r 的 left thread、right thread 以及原 successor 的 predecessor thread。
考試容易錯：Thread 不是 child；必須靠 leftThread / rightThread 判斷指標意義。
