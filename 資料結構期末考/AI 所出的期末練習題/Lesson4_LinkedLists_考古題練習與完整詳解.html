題目原文

Given an ordered list stored by an array and by a singly linked list:
EAT, FAT, HAT, WAT

a. Insert "GAT" between FAT and HAT.
b. Delete "GAT" after insertion.
c. Compare the time complexity of insertion and deletion in the middle of an array and a linked list, assuming the target position is already known.

中文翻譯

給定一個 ordered list，分別用陣列與單向 linked list 表示。請插入 GAT、刪除 GAT，並比較在「已知目標位置」時，array 與 linked list 的中間插入、刪除時間複雜度。

核心觀念

array shiftinglinked insertionlinked deletionO(n)O(1)

動態圖解

解題步驟

  1. Array 插入中間時，HAT 與後面的元素需要右移，所以插入本身需要 O(n)。
  2. Array 刪除中間元素時，後面元素需要左移，所以刪除需要 O(n)。
  3. Linked list 若已知 FAT 的節點位置，只要改兩條 link 就可插入 GAT，所以 O(1)。
  4. Linked list 若已知 GAT 前一個節點 FAT，也只要把 FAT 的 link 改到 HAT，所以 O(1)。
表示法插入中間刪除中間原因
ArrayO(n)O(n)需要搬移元素
Linked listO(1)O(1)只改 pointer/link
完整答案：Array 在中間插入與刪除需要搬移元素，因此是 O(n)。Linked list 在已知目標位置或前一節點時，只需改 link，因此插入與刪除都是 O(1)。
考試容易錯：如果「不知道目標位置」，linked list 仍需從 first 找到該位置，搜尋本身是 O(n)。本題已假設 target position known，所以只比較插入／刪除動作。

題目原文

Define a C++ node class for a singly linked list of integers.
Then draw a linked list containing 10 -> 20 -> 30 -> NULL.

中文翻譯

請定義一個整數單向 linked list 的節點類別，並畫出 10 → 20 → 30 → NULL 的 linked list。

核心觀念

nodedatalinkNULLfirst pointer

圖解

完整 C++ 程式碼

#include <iostream>
using namespace std;

/*
 * ChainNode：單向 linked list 的節點
 * data：存放節點資料
 * link：指向下一個節點；若為 nullptr，代表串列結尾
 */
class ChainNode {
public:
    int data;             // 節點資料
    ChainNode* link;      // 下一個節點的指標

    // 建構子：建立節點時可同時設定資料與下一個節點
    ChainNode(int element = 0, ChainNode* next = nullptr) {
        data = element;
        link = next;
    }
};

int main() {
    // 建立 10 -> 20 -> 30 -> NULL
    ChainNode* third = new ChainNode(30, nullptr);
    ChainNode* second = new ChainNode(20, third);
    ChainNode* first = new ChainNode(10, second);

    // 輸出 linked list
    for (ChainNode* p = first; p != nullptr; p = p->link) {
        cout << p->data << " ";
    }

    // 釋放記憶體
    delete third;
    delete second;
    delete first;
    return 0;
}

函式與語法說明

ChainNode
表示 linked list 的一個節點。
data
存放節點中的資料。
link
指向下一個節點，形成鏈結。
nullptr
表示沒有下一個節點，也就是 linked list 結尾。
new
在 heap memory 中動態建立節點。
完整答案：一個 singly linked list node 至少包含 data 與 link。first 指向第一個節點，最後一個節點的 link 指向 NULL/nullptr。
考試容易錯：不要只寫 int data；linked list 的重點是節點必須有 link pointer 才能串起下一個節點。

題目原文

Explain why ChainNode is usually declared as a friend of Chain.
Also explain the meaning of "Chain HAS-A ChainNode".

中文翻譯

請解釋為什麼 ChainNode 通常會讓 Chain 成為 friend class，並說明「Chain HAS-A ChainNode」的意思。

核心觀念

friend classprivate dataencapsulationHAS-A

圖解說明

解題步驟

  1. ChainNode 的 data/link 通常設為 private，避免外部隨便改壞 linked list。
  2. Chain 需要執行 insert/delete 等 list 操作，因此必須存取 ChainNode 的 data/link。
  3. 使用 friend class Chain，可以讓 Chain 存取 ChainNode 的 private members，但一般外部程式仍不能任意存取。
  4. HAS-A 表示 Chain 概念上包含很多 ChainNode；實作上通常是 Chain 內有一個 first pointer 指向第一個 ChainNode。
完整答案：ChainNode 使用 friend class Chain，是為了讓 Chain 的成員函式能直接操作節點的 private data 與 link，同時維持封裝性。Chain HAS-A ChainNode 表示 Chain 是由節點組成的資料結構，實作上 Chain 會有 ChainNode* first 指向第一個節點。
考試容易錯：friend 不是繼承，不是 IS-A；Chain 與 ChainNode 是 HAS-A / composite 關係。

題目原文

Given a linked list:
first -> 10 -> 20 -> 30 -> NULL

a. Insert a new node with data 50 after the node containing 20.
b. Delete the node containing 20.
c. Show the linked list after each operation and write the key pointer assignments.

中文翻譯

給定 linked list：first → 10 → 20 → 30 → NULL。請在 20 後插入 50，接著刪除 20，並寫出重要的 pointer assignment。

核心觀念

insert after xdelete nodepredecessorpointer assignment

動態圖解

完整 C++ 程式碼

/*
 * insertAfter(x, value)
 * 功能：在節點 x 後面插入一個新節點。
 * 關鍵：新節點先指向 x 原本的下一個節點，再讓 x 指向新節點。
 */
void insertAfter(ChainNode* x, int value) {
    if (x == nullptr) return;
    ChainNode* newNode = new ChainNode(value, x->link);
    x->link = newNode;
}

/*
 * deleteAfter(prev)
 * 功能：刪除 prev 後面的節點。
 * 關鍵：先保存要刪除的節點，再讓 prev 跳過它，最後 delete。
 */
void deleteAfter(ChainNode* prev) {
    if (prev == nullptr || prev->link == nullptr) return;
    ChainNode* delNode = prev->link;
    prev->link = delNode->link;
    delete delNode;
}

解題步驟

  1. 插入 50 after 20：先讓 50 指向 20 原本的下一個節點 30。
  2. 再讓 20 的 link 指向 50。
  3. 刪除 20 時，必須知道 20 的前一個節點 10。
  4. 讓 10 的 link 指向 20 的下一個節點 50，再 delete 20。
完整答案：插入後：10 → 20 → 50 → 30 → NULL。刪除 20 後：10 → 50 → 30 → NULL。重要指令：newNode->link = x->link; x->link = newNode; 刪除時 y->link = x->link; delete x。
考試容易錯：插入時順序不能反。如果先做 x->link = newNode，會遺失原本的下一個節點 30。

題目原文

Assume an iterator p points to the node containing 10 in the chain:
10 -> 20 -> 30 -> NULL

a. What is the value returned by *p?
b. After executing ++p, which node does p point to?
c. Explain the difference between preincrement (++p) and postincrement (p++).

中文翻譯

假設 iterator p 指向 10。請回答 *p 的值、++p 後 p 指向哪裡，並解釋 preincrement 與 postincrement 差異。

核心觀念

iteratoroperator*++pp++return by reference/value

動態圖解

完整 C++ 程式碼

template <class T>
class ChainIterator {
public:
    // 建構子：讓 iterator 指向某一個節點
    ChainIterator(ChainNode<T>* startNode = nullptr) {
        current = startNode;
    }

    // 解參照：取得目前節點中的資料
    T& operator*() const {
        return current->data;
    }

    // preincrement：先移動，再回傳移動後的自己
    ChainIterator& operator++() {
        current = current->link;
        return *this;
    }

    // postincrement：先保留舊狀態，再移動，最後回傳舊狀態
    ChainIterator operator++(int) {
        ChainIterator old = *this;
        current = current->link;
        return old;
    }

private:
    ChainNode<T>* current; // iterator 目前指向的節點
};

解題步驟

  1. *p 表示取得目前 iterator 指向節點的 data，所以是 10。
  2. ++p 會先移動到下一個節點，因此 p 指向 20。
  3. p++ 會回傳移動前的舊 iterator，但 p 本身最後仍會移到下一個節點。
完整答案：a. *p = 10。b. ++p 後 p 指向 20。c. ++p 先遞增再回傳目前物件；p++ 先保存舊物件，遞增後回傳舊物件。
考試容易錯：p++ 與 ++p 最後 p 都會往下一個節點移動；差別在「回傳的是舊狀態還是新狀態」。

題目原文

Given a chain:
first -> 1 -> 2 -> 3 -> 4 -> NULL

a. Show the result after reversing the chain in-place.
b. During reverse, explain the roles of current, previous, and r.
c. What is the time complexity?

中文翻譯

給定 linked list：1 → 2 → 3 → 4 → NULL。請做 in-place reverse，說明 current、previous、r 的角色，並回答時間複雜度。

核心觀念

reversein-placecurrentpreviousO(n)

Reverse 動態圖解

完整 C++ 程式碼

/*
 * Reverse()
 * 功能：將 linked list 原地反轉。
 * current：目前正在處理的節點
 * previous：已反轉區段的第一個節點
 * r：暫存 previous 的舊值，用來讓 current 反向連回去
 */
void Chain::Reverse() {
    ChainNode* current = first;
    ChainNode* previous = nullptr;

    while (current != nullptr) {
        ChainNode* nextNode = current->link; // 先保存下一個節點，避免斷鏈
        current->link = previous;            // 將目前節點的 link 反向
        previous = current;                  // previous 往前推進
        current = nextNode;                  // current 移到下一個未處理節點
    }

    first = previous; // previous 最後會指向新的第一個節點
}

解題步驟

  1. 一開始 previous = NULL，current 指向 1。
  2. 每次把 current 的 link 改成 previous，形成反向連結。
  3. current 繼續往原本下一個節點移動。
  4. 最後 first 指向 previous，也就是原本最後一個節點 4。
完整答案：反轉後為 4 → 3 → 2 → 1 → NULL。current 表示目前處理節點，previous 表示已反轉部分的開頭，r 或 nextNode 用來暫存指標避免斷鏈。時間複雜度 O(n)，額外空間 O(1)。
考試容易錯：一定要先保存下一個節點，不然 current->link 一反向後，原本後面的 list 會遺失。

題目原文

Assume a circular list uses a pointer last pointing to the last node.
The list is:
last -> C, and C.link points to A
A -> B -> C -> A

a. Insert X at the front of the circular list.
b. Explain why using last is more convenient than using first.
c. What is the purpose of a header node?

中文翻譯

假設 circular list 使用 last pointer 指向最後一個節點。請將 X 插入前端，說明為什麼用 last 比 first 方便，以及 header node 的用途。

核心觀念

circular listlast pointerInsertFrontheader node

動態圖解

完整 C++ 程式碼

/*
 * InsertFront(e)
 * 功能：在 circular list 的前端插入節點。
 * 假設 last 指向最後一個節點，last->link 就是第一個節點。
 */
void CircularList::InsertFront(int e) {
    ChainNode* newNode = new ChainNode(e);

    if (last != nullptr) {
        // newNode 先指向原本第一個節點
        newNode->link = last->link;

        // last 的 link 改成 newNode，使 newNode 成為新的 first
        last->link = newNode;
    } else {
        // 空串列：新節點自己指向自己
        last = newNode;
        newNode->link = newNode;
    }
}

解題步驟

  1. 在 circular list 中，last->link 就是 first。
  2. 插入 X 到 front 時，先令 X->link = last->link。
  3. 再令 last->link = X。
  4. last 不需要改，因為最後一個節點仍是 C。
  5. Header node 可以統一空串列與非空串列的處理，減少特殊情況。
完整答案：插入後為 X → A → B → C → X。使用 last 方便是因為 last->link 直接就是 first，所以可 O(1) 在前端插入。Header node 是 dummy node，可簡化 empty list 的邊界處理。
考試容易錯：InsertFront 不需要改 last；InsertBack 才需要把 last 更新成 newNode。

題目原文

a. Use a linked stack to perform Push(10), Push(20), Pop().
b. Use a linked queue to perform Push(10), Push(20), Pop().
c. Show the final structures and explain the difference.

中文翻譯

請分別用 linked stack 與 linked queue 執行 Push(10)、Push(20)、Pop()，畫出最後結果並說明差異。

核心觀念

stack LIFOqueue FIFOtopfrontrear

動態圖解

完整 C++ 程式碼

/*
 * LinkedStack::Push
 * 新節點插入 top 前面，讓 top 指向新節點。
 */
void LinkedStack::Push(int e) {
    top = new ChainNode(e, top);
}

/*
 * LinkedStack::Pop
 * 刪除 top 節點，top 往下一個節點移動。
 */
void LinkedStack::Pop() {
    if (top == nullptr) throw "Stack is empty";
    ChainNode* delNode = top;
    top = top->link;
    delete delNode;
}

/*
 * LinkedQueue::Push
 * 新節點接在 rear 後面。
 */
void LinkedQueue::Push(int e) {
    if (front == nullptr) {
        front = rear = new ChainNode(e, nullptr);
    } else {
        rear->link = new ChainNode(e, nullptr);
        rear = rear->link;
    }
}

/*
 * LinkedQueue::Pop
 * 刪除 front 節點。
 */
void LinkedQueue::Pop() {
    if (front == nullptr) throw "Queue is empty";
    ChainNode* delNode = front;
    front = front->link;
    if (front == nullptr) rear = nullptr;
    delete delNode;
}

解題步驟

  1. Stack：Push(10) 後 top=10；Push(20) 後 top=20→10；Pop() 刪除 20，剩 10。
  2. Queue：Push(10) 後 front=rear=10；Push(20) 後 10→20；Pop() 刪除 front 10，剩 20。
  3. Stack 是 LIFO，後進先出；Queue 是 FIFO，先進先出。
完整答案：Linked stack 最後剩 10。Linked queue 最後剩 20。差異是 stack 刪除最新加入的節點，queue 刪除最早加入的節點。
考試容易錯：Queue 在刪除最後一個節點時，front 與 rear 都要變成 NULL；否則 rear 會變成 dangling pointer。

題目原文

Explain the purpose of an available space list (av list).
Assume av points to a chain of deleted nodes.
a. What does GetNode() do?
b. What does RetNode(x) do?
c. Why can deleting a whole circular list be done in O(1) if both first and last are known?

中文翻譯

請解釋 available space list 的用途。av 指向已刪除節點串列，請說明 GetNode()、RetNode(x)，以及為什麼 circular list 若知道 first 與 last 可 O(1) 刪除整串。

核心觀念

av listrecycle nodesGetNodeRetNodeO(1)

動態圖解

完整 C++ 程式碼

/*
 * av：available space list 的開頭指標
 * 用來保存已刪除但可重複使用的節點。
 */
static ChainNode* av = nullptr;

/*
 * GetNode()
 * 功能：取得一個可用節點。
 * 若 av 不空，從 av list 拿一個舊節點；否則才 new 新節點。
 */
ChainNode* GetNode() {
    ChainNode* x;
    if (av != nullptr) {
        x = av;
        av = av->link;
    } else {
        x = new ChainNode();
    }
    return x;
}

/*
 * RetNode(x)
 * 功能：把不再使用的節點 x 放回 av list。
 */
void RetNode(ChainNode*& x) {
    x->link = av;
    av = x;
    x = nullptr;
}

解題步驟

  1. Available space list 把刪除後的節點串起來，之後可重複使用。
  2. GetNode() 優先從 av 拿節點，減少 new 的成本。
  3. RetNode(x) 把 x 接到 av 前端。
  4. 若 circular list 知道 first 與 last，可直接 last->link = av; av = first; last = NULL，把整串接到 av list 前面，不用逐一 delete。
完整答案：av list 是一條回收節點串列。GetNode() 從 av 取出可用節點或 new 新節點；RetNode(x) 把 x 放回 av。若 circular list 的 first、last 已知，可用常數次 pointer assignment 把整串接到 av，因此 O(1)。
考試容易錯：av list 不是垃圾回收器；它是程式自己維護的節點重用機制。

題目原文

Two polynomials are represented by linked lists in descending order of exponent.

A(x) = 3x^14 + 2x^8 + 1
B(x) = 8x^14 - 3x^10 + 10x^6

a. Draw the linked-list representation.
b. Compute C(x) = A(x) + B(x).
c. What is the time complexity if A has m terms and B has n terms?

中文翻譯

兩個多項式用 linked list 依指數由大到小儲存。請畫出表示法、計算 A+B，並回答時間複雜度。

核心觀念

polynomialcoefexpmergeO(m+n)

Polynomial Addition 動態圖解

完整 C++ 程式碼

struct Term {
    int coef; // 係數
    int exp;  // 指數
};

/*
 * Polynomial::Add
 * 功能：將兩個已依 exp 由大到小排序的多項式相加。
 * ai：指向 A 的目前項
 * bi：指向 B 的目前項
 * c：結果多項式
 */
Polynomial Polynomial::Add(const Polynomial& b) const {
    Iterator ai = poly.begin();
    Iterator bi = b.poly.begin();
    Polynomial c;

    while (ai != poly.end() && bi != b.poly.end()) {
        if (ai->exp == bi->exp) {
            // 指數相同：係數相加
            int sum = ai->coef + bi->coef;
            if (sum != 0) c.InsertBack({sum, ai->exp});
            ++ai;
            ++bi;
        } else if (ai->exp < bi->exp) {
            // B 的指數較大，先複製 B 的目前項
            c.InsertBack(*bi);
            ++bi;
        } else {
            // A 的指數較大，先複製 A 的目前項
            c.InsertBack(*ai);
            ++ai;
        }
    }

    // 複製 A 剩餘項
    while (ai != poly.end()) {
        c.InsertBack(*ai);
        ++ai;
    }

    // 複製 B 剩餘項
    while (bi != b.poly.end()) {
        c.InsertBack(*bi);
        ++bi;
    }

    return c;
}

解題步驟

  1. A 的節點為 (3,14) → (2,8) → (1,0)。
  2. B 的節點為 (8,14) → (-3,10) → (10,6)。
  3. 14 與 14 相同，係數相加：3+8=11，所以得到 11x^14。
  4. B 的 10 比 A 的 8 大，copy -3x^10。
  5. A 的 8 比 B 的 6 大，copy 2x^8。
  6. B 剩下 10x^6，A 剩下 1，依序 copy。
完整答案：C(x)=11x^14−3x^10+2x^8+10x^6+1。若 A 有 m 項、B 有 n 項，每個節點最多走訪一次，所以時間複雜度 O(m+n)。
考試容易錯：比較的是 exponent，不是 coefficient；指數相同才做係數相加。

Represent the following 5 x 4 sparse matrix using the linked representation with header nodes.

[ 2 0 0 0
  4 0 0 3
  0 0 0 0
  8 0 0 1

請用含 header node 的 linked representation 表示 5×4 稀疏矩陣。列出非零三元組、總節點數，並說明 header node 與 element node 差異。

sparse matrixtripleheader nodeelement nodemax(n,m)+r+1

圖解說明

  4. 還有一個總 header node H，所以總節點數是 max(n,m)+r+1。
Node type欄位用途
Header nodehead, down, right, next管理 row list、column list 與 header list
Element nodehead, row, col, value, down, right表示一個非零元素，並同時串在 row 與 column 中
完整答案：非零 triples 為 (0,0,2)、(1,0,4)、(1,3,3)、(3,0,8)、(3,3,1)、(4,2,6)。總節點數=max(5,4)+6+1=12。
考試容易錯：總節點數不是 n×m，也不是 r；linked sparse matrix 還需要 header nodes 與總 head node。

題目原文

Given a doubly linked circular list with a header node:
head <-> 10 <-> 20 <-> head

a. Insert node 15 to the right of node 10.
b. Delete node 20.
c. Show the pointer assignments for insertion and deletion.

中文翻譯

給定含 header node 的雙向環狀串列 head ↔ 10 ↔ 20 ↔ head。請在 10 右邊插入 15，然後刪除 20，並寫出指標更新。

核心觀念

doubly linked listleft linkright linkheader nodepointer order

動態圖解

完整 C++ 程式碼

class DblListNode {
public:
    int data;
    DblListNode* left;
    DblListNode* right;
};

/*
 * Insert(p, x)
 * 功能：將節點 p 插入到節點 x 的右邊。
 */
void DblList::Insert(DblListNode* p, DblListNode* x) {
    p->left = x;             // p 的左邊是 x
    p->right = x->right;     // p 的右邊是 x 原本的右節點
    x->right->left = p;      // 原本右節點的 left 改成 p
    x->right = p;            // x 的 right 改成 p
}

/*
 * Delete(x)
 * 功能：刪除節點 x。
 * 注意：通常不能刪除 header node。
 */
void DblList::Delete(DblListNode* x) {
    if (x == first) throw "Please do not delete the header node";

    x->left->right = x->right;  // 左邊節點跳過 x
    x->right->left = x->left;   // 右邊節點跳過 x
    delete x;
}

解題步驟

  1. 插入 15 到 10 右邊時，15 的 left 指向 10。
  2. 15 的 right 指向 10 原本的 right，也就是 20。
  3. 20 的 left 改成 15。
  4. 10 的 right 改成 15。
  5. 刪除 20 時，讓 15 的 right 指向 head，head 的 left 指向 15。
完整答案：插入後：head ↔ 10 ↔ 15 ↔ 20 ↔ head。刪除 20 後：head ↔ 10 ↔ 15 ↔ head。插入需 4 個 pointer assignments，刪除需 2 個 pointer assignments。
考試容易錯：雙向串列插入時左右兩邊都要接好；只改 right 不改 left，串列會壞掉。
